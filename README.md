# Система анализа корректности произношения (ВКР)

## О проекте

Этот проект — исследовательская ВКР по теме автоматизации языкового дриллинга.

Выполнен Муратовым А.А.

Основная идея: реализовать и сравнить два подхода к оценке произношения:
1. Классические алгоритмы -  DTW и MFCC.
2. Нейросети - Wave2Vev 2.0.

Сейчас ИИ добавляют во многие алгоритмы и мне хотелось бы проверить, есть ли разница в точности между классикой и современным подходом. Кроме того, выбор подхода имеет прямое влияние на продакшн приложения с подобным функционалом. Классические алгоритмы требуют мало ресурсов, а потому могут размещаться непосредственно на пользовательском устройстве. Нейросети не могут таких похвастаться - они требуют больших ресурсов, которые потребуется размещать на отдельном сервере.

## Цель исследования

Сравнить подходы по ключевым критериям:

- качество оценки произношения
- устойчивость к шуму и разным условиям записи
- вычислительные затраты (скорость, ресурсы)
- интерпретируемость результата для пользователя
